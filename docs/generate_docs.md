# Інструкція з генерації документації (DocC)

Цей проєкт використовує нативний компілятор Apple **DocC** для генерації документації.

## Генерація через інтерфейс Xcode (Локально)
1. Відкрийте проєкт `WeatherBrick.xcodeproj` у Xcode.
2. У верхньому меню оберіть **Product** -> **Build Documentation** (або натисніть `Ctrl + Shift + Cmd + D`).
3. Xcode автоматично збере код, згенерує документацію та відкриє її у вікні Developer Documentation.
4. Щоб експортувати документацію у вигляді архіву (для передачі іншим), у вікні документації натисніть правою кнопкою миші на назву проєкту на панелі навігатора зліва і оберіть **Export**. Файл буде збережено з розширенням `.doccarchive`.

## Генерація через термінал (xcodebuild)
Для генерації документації без графічного інтерфейсу (наприклад, для CI/CD) використовуйте команду:
\`\`\`bash
xcodebuild docbuild -scheme WeatherBrick -destination "platform=iOS Simulator,name=iPhone 15 Pro"
\`\`\`
